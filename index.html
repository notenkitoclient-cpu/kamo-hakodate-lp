講演内容

  • ✔ 普通の会社員から「プロ会社員」へアップデートする方法
  • ✔ 赤字社員から黒字社員へ変わる視点
  • ✔ 部下をどう育て、チームを成長させる承認の力

といった、ビジネスの根本的な考え方から、

プロ会社員になるための具体的な戦略

自分の給料に見合う以上の成果を出すために、何をどう考え、どんな行動をすればよいのかを明確に解説します。

会社に依存しない働き方の実践方法

「どうせ会社員だから…」という思い込みを壊し、自立した働き方を身につけ、プロとしての誇りを持つための行動プランを共有します。

承認力でチームを成長させる秘訣

部下や仲間の心のエネルギーを高め、組織全体の成果を劇的に伸ばす承認の技術を徹底解説。

この講演を通して、あなたは「私は会社員です」ではなく
「私はプロ会社員です」と胸を張って名乗れるようになります。

こんな方にオススメ

  • ✔ 今の働き方に限界を感じ、「もっと成長したい」と思っている会社員の方
  • ✔ 給料や評価に不満を抱きながらも現状を変えたい会社員の方
  • ✔ 部下をどう育て、組織をまとめればいいか悩んでいる管理職の方
  • ✔ 人材不足・離職率の高さに頭を抱えている経営者やリーダーの方
  • ✔ チームのモチベーションを高め、成果につなげたいと願う管理職の方
  • ✔ 経営者として、社員が主体的に動き業績を伸ばす組織をつくりたい方
  • ✔ 自社の価値をもっと上げたいと思っている経営者の方
  • ✔ 時代の変化に負けない柔軟な思考を身に付けたい方
  • ✔ これからの時代に必要な“プロ会社員”のマインドとスキルを身につけたい全てのビジネスパーソン

株式会社東京カモガシラランド　代表取締役社長

日本マクドナルドでのキャリアを経て、2010年に独立し、株式会社東京カモガシラランドを設立。全国3300店舗中で「顧客満足度」「従業員満足度」「セールス伸び率」の三冠を達成した実績を基に、組織マネジメントや人材育成に特化した講演活動を全国で展開。これまでに3000回以上の講演を行い、累計30万人以上を動員している。

経営者としても多方面で成功を収め、高級焼肉店「YAKINIKUMAFIA IKEBUKURO」を1店舗、神戸牛鉄板焼専門店「和牛の刀」を2店舗運営。特にインバウンド市場を意識した高級食文化ブランドの構築により、日本の食文化を国内外に発信する活動を行っている。
2025年1月1日には、速読スクール「ビジネス脳トレ速読」を立ち上げ、ビジネスパーソンを対象に速読を通じて思考力や判断力を高める革新的な教育プログラムを展開予定。

著者としても精力的で、リーダーや経営者に向けた実践的な内容をテーマに24冊（海外2冊含む）の書籍を出版。
YouTubeでは登録者数100万人超え、総再生回数3億回以上という圧倒的な影響力を持ち、SNSフォロワー数は250万人以上。挑戦する力と可能性を多くの人々に届ける自己啓発業界のリーダーとして活躍している。
